CHANGES FROM ORIGINAL SPECIFICATION
-------------------------------------
Lag selection updated:
  - Changed from lag=10 (unjustified) to lag=6
  - Based on Newey-West (1994) formula: 4 * (T/100)^(2/9)
  - For T ≈ 600 weekly observations: 4 * (6)^(2/9) ≈ 5.9 → lag = 6
  - Still conservative enough to handle Bitcoin's known volatility clustering

Controls in primary models (Models 1 & 2): unchanged from original.
  DXY and 10Y yield moved to Model 3 (robustness only) — see rationale below.

============================================================

MODEL 1 — Full-Sample Interaction Model (Primary Test)

R_BTC_t = α
  + β₁ · dFedRate_t
  + β₂ · dFedBS_t
  + β₃ · HalvingWindow_t
  + β₄ · PostETF_t
  + β₅ · (dFedRate_t × PostETF_t)
  + β₆ · (dFedBS_t × PostETF_t)
  + β₇ · (HalvingWindow_t × PostETF_t)
  + γ₁ · R_SP500_t
  + γ₂ · R_BTC_lag1
  + γ₃ · dVIX_t
  + ε_t

Estimation: OLS with Newey-West HAC standard errors
            (Bartlett kernel, lag = 6)

Hypothesis interpretation:
  H1: β₅ and β₆ — if significant and same-signed as β₁/β₂,
      macro sensitivity strengthened post-ETF.
  H2: β₇ — if β₃ > 0 (halving boosts returns) but β₇ < 0 or
      insignificant, the halving effect diminished post-ETF.
      Total post-ETF halving effect = β₃ + β₇.

Expected signs:
  β₁ (dFedRate)   : negative (rate hikes suppress BTC)
  β₂ (dFedBS)     : positive (Fed expansion = liquidity = BTC up)
  β₃ (HalvingWin) : positive (supply shock historically bullish)
  γ₁ (R_SP500)    : positive (risk-on correlation)
  γ₃ (dVIX)       : negative (fear = risk-off = BTC down)

============================================================

MODEL 2 — Sub-Sample Split (Robustness Check)

Run separately on:
  A) Pre-ETF:  July 2014 – December 2023  (~490 weeks)
  B) Post-ETF: January 2024 – latest      (~110+ weeks)

R_BTC_t = α
  + β₁ · dFedRate_t
  + β₂ · dFedBS_t
  + β₃ · HalvingWindow_t
  + γ₁ · R_SP500_t
  + γ₂ · R_BTC_lag1
  + γ₃ · dVIX_t
  + ε_t

Estimation: OLS with Newey-West HAC standard errors (lag = 6)

Compare across sub-samples:
  - R²           : B > A supports H1 (macro explains more post-ETF)
  - β₁, β₂      : larger in B supports H1
  - β₃           : smaller/insignificant in B supports H2
  - γ₁ (S&P500) : larger in B = bonus finding on equity integration

Caveat: Sub-sample B has ~110 observations. Use as directional
        robustness, not standalone proof.
